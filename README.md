Welcome to the repository for my personal portfolio! This project is a premium, dark-themed web application designed to showcase my journey as a developer, highlighting my projects, technical stack, and professional experience through a highly interactive and cinematic user interface.

## 🚀 Experience the Flow

The site is built with a heavy focus on performance, smooth transitions, and a modern aesthetic:
- **Intro Preloader**: A word-reveal animation that transitions seamlessly into the main content.
- **Dynamic Hero Section**: A responsive profile card featuring real-time availability indicators and key layout statistics.
- **Diagonal-Scrolling Project Gallery**: A unique, scroll-driven parallax gallery that slides diagonally on the screen as you explore my featured programming work.
- **3D Tech Sphere & Skills**: An interactive 3D icon cloud grouped alongside permanently highlighted category tags that dynamically map transparency and glow using modern CSS.
- **Interactive Timeline**: A vertical work experience tracker built to outline my career history and academic achievements.
- **Floating Dock Navigation**: A pill-shaped, interactive navbar with scroll-spy active state tracking.
- **Express Backend Integration**: A Node.js API infrastructure to handle contact form submissions and local processes.

## 🛠️ Tech Stack

**Frontend Architecture:**
- **Core Engine**: React 19, Vite, Context APIs
- **Styling**: Vanilla CSS leveraging modern features like Custom Variables, grid systems, `color-mix()` for dynamic transparency, and high-end glassmorphism.
- **Animation Framework**: *Framer Motion* for intricate scroll-driven parallax movements (`useScroll`, `useTransform`, `useInView`), and *React Icon Cloud* for 3D visualizations.
- **Typography & Assets**: Lucide React Icons, Outfit typography suite.

**Backend Setup:**
- **Environment**: Node.js, Express
- **Middlewares**: CORS, Dotenv

## 🏁 Getting Started

If you'd like to check out the code locally, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Frontend Setup
```bash
# Clone the repository and navigate to the project root
npm install
npm run dev
```
The client application will boot up locally at `http://localhost:5173/`.

### 3. Backend Setup
*(Required to test the local Contact form API)*
```bash
# Navigate to the server directory
cd server
npm install
node index.js
```
The API server will run on `http://localhost:5000/`.
